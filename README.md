# AI-Powered Career Guidance Web Application

## Overview

AI-Powered Career Guidance is a full-stack web application designed to help students make informed academic and career decisions through personalized recommendations and intelligent insights. The platform combines artificial intelligence, career assessment, and comprehensive educational resources to guide users from stream selection to long-term career planning.

## Features

### AI Career Assessment

* Interactive career assessment to identify users' interests, strengths, and aptitude.
* Personalized career recommendations based on assessment results.

### AI Career Assistant

* Integrated AI chatbot powered by **Flan-T5** and **Hugging Face Transformers**.
* Answers career-related questions, provides guidance, and suggests personalized learning roadmaps.

### Career & Course Recommendations

* Maps career paths to relevant degrees, certifications, and skills.
* Recommends suitable courses for different career goals.

### College & Institution Finder

* Browse colleges and institutions with important details, rankings, and available programs.
* Helps students compare educational opportunities.

### Career Insights

* Explore salary trends, required skills, career growth, and industry outlook across multiple domains.

### User-Friendly Interface

* Responsive design for seamless experience across desktop and mobile devices.
* Simple navigation and intuitive user experience.
